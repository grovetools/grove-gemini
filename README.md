# grove-gemini

Tools for Google's Gemini API

## Installation

```bash
grove install gemapi
```

## Usage

```bash
gemapi --help
```

## Development

### Building

```bash
make build
```

### Testing

```bash
make test
make test-e2e
```

### Linting

```bash
make lint
```
